## ASP.NET Core Web API Application

- This project serves as the backend API for a React application.

- It provides full CRUD functionality through RESTful endpoints using HTTP methods such as GET, POST, PUT, and DELETE.
  
- Frontend React Application: https://github.com/Divyadevaki/CRUD_ReactViteApp.git

## HTTP GET – Retrieve Data

- Used to fetch or read data from the server.
- Does not modify any existing data.

## HTTP POST – Create New Data

- Used to insert or create new records on the server.
- The data required for creating a resource is sent in the request body.

## HTTP PUT – Update Existing Data

- Used to update or modify an existing record.
- Typically requires the resource ID and updated data in the request body.

## HTTP DELETE – Remove Data

- Used to delete or remove a specific record from the server.
- Requires the resource ID to identify which record to delete.
